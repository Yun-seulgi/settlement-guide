⬡ 외주 정산센터 사용 가이드

월별 외주 정산 업무 처리 가이드

업체 리스트 상세내역 외주사 비교 저장 구조 카톡 공지문 주의사항 Q&A
1
좌측 업체 리스트
✅ 공지 / 확정 / 계산서 체크
각 업체 행의 체크박스 3개로 진행 상태를 관리합니다.
공지 카톡 공지 완료   확정 금액 확정 완료   계산서 세금계산서 수령 완료

3개 모두 체크되면 해당 업체 행이 연한 초록색으로 변합니다.
열 헤더를 클릭하면 전체 업체를 일괄 체크/해제할 수 있습니다.
💰 확정금액
업체별 최종 정산 금액을 입력하는 칸입니다. 숫자 입력 시 콤마 자동 포함.

상단 확정금액 ⚡ 열 헤더를 클릭하면 모든 업체의 기본 정산금액이 확정금액 칸에 한 번에 채워집니다. (지방외주는 정산금액 자동 복사, 수도권 수기는 입력되어 있던 금액 기준)

확정금액을 입력하고 확정 체크를 켜면, 외주사 비교 탭의 기준 금액이 확정금액으로 바뀝니다.
📤 하단 복사 버튼
업체명 복사 건수 복사 확정금액 복사

전체 업체 데이터를 줄바꿈 형식으로 클립보드에 복사합니다. 엑셀에 바로 붙여넣기 가능합니다.
🟠 수도권 업체 추가
수도권 업체의 건수/금액은 각 업체의 시트에서 불러오기 됩니다. 리스트 하단 입력창에서 업체명을 직접 입력하거나, 엑셀에서 업체명 / 건수 / 금액 열을 복사해 대량 붙여넣기(Ctrl+V)할 수 있습니다.

수도권 업체는 마감 저장 버튼을 눌러야 시트에 반영됩니다. 삭제는 업체명 옆 ✕ 버튼으로 가능합니다. (수도권은 수정할일 없긴할듯!)
💡
초록색 합계 금액 클릭 시 '순수 숫자 복사' 업체 리스트에 표시된 정산금액 숫자를 마우스로 클릭하면, 콤마(,)가 자동으로 제외된 순수 숫자가 클립보드에 즉시 복사됩니다. 확정금액이나 다른곳에 금액을 수기 입력할 때 활용하세요.
2
📋 상세내역 탭
기안용 복사
📎 기안용 복사 버튼을 누르면 해당 업체의 건별 내역 전체가 탭 구분 형식으로 복사됩니다. 엑셀이나 결재 문서에 바로 붙여넣기 가능합니다.
계산서 요청 문자 복사
상단 합계금액 영역을 클릭하면 해당 업체에 보낼 세금계산서 요청 문자가 자동 생성되어 복사됩니다.
⚠️
기안용 복사 전 반드시 확인 상세내역에는 구글 시트에서 완료(TRUE) 처리된 건만 불러옵니다. 기안서 작성 전에 해당 월 말일까지의 모든 건이 시트에서 완료 처리되어 있는지 먼저 확인하세요. 완료 처리가 안 된 건은 표시되지 않습니다.
3
⚖️ 외주사 합계 비교 탭
외주사 내역 입력 방법
외주사에서 받은 정산 내역 엑셀에서 상호명 / 금액 2열을 복사한 뒤, 상단 붙여넣기 입력창에 Ctrl+V하면 오른쪽 패널에 자동으로 행이 채워집니다.
자동 매칭
금액과 업체명이 일치하는 건은 초록색으로 자동 표시됩니다. 시스템이 자체적으로 상호명 속 노이즈 단어(‘(주)’, ‘주식회사’ 등)를 제거하고 유사도 검사를 실행합니다.

텍스트가 다른것들은 좌측 패널에서 수동으로 [일치] 체크박스를 직접 클릭해 처리할 수 있습니다.

상단에 아샵 합계(A) / 외주 청구(B) / 차액(A-B)이 실시간으로 표시되며, 차액이 0원이면 ✓ 금액 일치로 바뀝니다.
아샵 데이터 수정
왼쪽 패널에서 금액을 직접 클릭해 수정하거나 메모를 추가할 수 있습니다. 수정 내역이 발생한 행은 주황색 배경으로 실시간 변환됩니다.

수정 후 💾 아샵 데이터 일괄 저장을 눌러야 스프레드시트 데이터에 반영됩니다.

이 버튼을 누르지 않으면 시트에 반영되지 않습니다.
⌨️
키보드 Enter 단축키 안내 (외주사 내역 입력 패널) 수기 입력 창에서 상호명이나 금액을 작성한 뒤 Enter를 누르면 아래 행의 입력창으로 포커스가 자동 이동합니다. 특히 마지막 행의 상호명 칸에서 엔터를 누르면 새로운 입력 행이 자동으로 추가되어 마우스 없이 빠른 타이핑이 가능합니다.
⚠️
외주사 비교 시점 주의 비교 작업은 보통 월말 마감 직전에 진행하므로, 이 시점에는 아직 완료 처리가 안 된 건이 존재할 수 있습니다. 외주사가 청구한 건이 아샵 데이터에 없다면 시트에서 완료 처리가 빠진 건인지 먼저 확인하세요.
💡
확정 체크 시 기준 금액 변경 좌측 리스트에서 해당 업체의 확정 체크를 켜면, 비교 기준이 아샵 합계가 아닌 확정금액으로 자동 전환됩니다. 확정금액을 별도로 입력한 경우에 활용하세요.
4
저장 구조
💾 마감 저장 → 정산마감_list 시트에 반영
체크박스(공지/확정/계산서) · 확정금액 · 메모 · 수도권 업체 목록이 저장됩니다.
지방외주설치건_list부분은 건드리지 않습니다.
💾 아샵 데이터 일괄 저장 → 지방외주설치건_list 시트, 수도권은 각 업체 시트에 반영
외주사 비교 탭에서 수정한 금액과 메모가 각 시트에 직접 덮어씌워집니다.
🔄 새로고침
시트의 최신 데이터를 다시 불러옵니다. 저장하지 않은 수정 내용은 모두 사라지니 저장 후 사용하세요.
5
카톡 공지문 복사
💬 사용 방법

상단 💬 카톡 공지 버튼을 누르면 해당 정산월 기준으로 작성된 공지문이 클립보드에 복사됩니다.

  • 설치 건 리스트 제출 요청 안내 문구 포함
  • 계산서 발행 시 유의사항 자동 포함 (말일 발행, 품목명, 발행 메일)
  • 월 정보가 자동으로 반영되어 매달 수정 불필요
6
주의사항
🔴
동시 접속 충돌 같은 달을 두 명이 동시에 수정하다 저장하면 마지막 저장 기준으로 덮어씌워집니다. 상단에 주황색 알림이 뜨면 저장 전 확인하고 작업하세요.
⚠️
완료 처리 기준 구글 시트에서 완료(TRUE) 체크가 된 건만 정산센터에 집계됩니다. 기안용 복사나 합계 금액이 맞지 않다면 시트의 완료 처리 여부를 먼저 확인하세요.
7
Q&A
Q. 외주사 엑셀을 붙여넣었는데 초록색 매칭이 안 되는 건이 있어요.
금액이나 상호명이 시스템 기준치 이상 다르면 자동 매칭이 스킵됩니다. 이럴 때는 아샵 데이터 우측 끝에 있는 [일치] 체크박스를 직접 클릭해 수동으로 매칭(초록색 전환)해 주시면 됩니다.
Q. 시트에서 완료(TRUE) 처리를 방금 했는데 정산센터에 안 보여요.
화면 상단 왼쪽의 [🔄 새로고침] 버튼을 누르면 구글 시트의 최신 데이터를 실시간으로 다시 긁어옵니다. 단, 임시로 화면에서 수정 중이던 내역은 사라지므로 주의하세요.
Q. 외주사 비교 창에서 아샵 금액을 고쳐서 주황색이 됐는데 원상복구 되나요?
'일괄 저장' 버튼을 누르기 전이라면 안전합니다. 아샵 정산 데이터 타이틀 옆의 [↺] (되돌리기) 버튼을 누르면 해당 업체의 데이터가 수정 전 원본 상태로 복구됩니다.
Q. [확정] 체크박스를 켰는데 비교 탭 상단 기준 금액(A)이 안 바뀌어요.
왼쪽'확정금액' 입력창에 0원보다 큰 최종 금액이 먼저 입력되어 있어야 작동합니다. 금액 칸을 먼저 채운 뒤 체크박스를 켜 주세요.
